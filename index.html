Materi Logaritma

  • Definisi: loga(b) = c berarti ac = b, dengan a > 0, a ≠ 1, b > 0.
  • Contoh 1: Hitung log3(x + 6) = 2
    1. Ubah ke bentuk eksponensial: 3² = x + 6
    2. 9 = x + 6 → x = 3
  • Contoh 2: Hitung log5(25x) = 3
    1. Ubah ke bentuk eksponensial: 5³ = 25x
    2. 125 = 25x → x = 5
  • Contoh 3: Hitung log(y − 1) 8 = 3
    1. Ubah ke bentuk eksponensial: (y − 1)³ = 8
    2. y − 1 = 2 → y = 3
  • Contoh 4: Hitung log2(4x − 8) = 5
    1. Ubah ke bentuk eksponensial: 2⁵ = 4x − 8
    2. 32 = 4x − 8 → 4x = 40 → x = 10
  • Contoh 5: Hitung log9(3y + 6) = 1/2
    1. Ubah ke bentuk eksponensial: 9^(1/2) = 3y + 6
    2. √9 = 3 → 3 = 3y + 6 → y = -1
  • Rumus penting:
    • loga(MN) = logaM + logaN
    • loga(M/N) = logaM − logaN
    • loga(Mⁿ) = n·logaM
    • logaa = 1
    • loga1 = 0

Materi SPLDV & SPLTV (Sistem Persamaan Linear)

  • Definisi SPLDV: Sistem persamaan linear dengan dua variabel (x dan y).
  • Definisi SPLTV: Sistem persamaan linear dengan tiga variabel (x, y, dan z).
  • Contoh SPLDV:

    Diberikan sistem:

    • 2x + 3y = 12
    • x − y = 1

    Metode Substitusi:

    1. Dari persamaan kedua: x = y + 1
    2. Substitusi ke persamaan pertama: 2(y + 1) + 3y = 12 → 5y + 2 = 12 → y = 2
    3. Maka x = y + 1 = 3

    Metode Eliminasi:

    1. Kalikan persamaan kedua agar y dapat dieliminasi: 2x − 2y = 2
    2. Kurangkan dari persamaan pertama: (2x + 3y) − (2x − 2y) = 12 − 2 → 5y = 10 → y = 2
    3. Maka x = 3
  • Contoh SPLTV:

    Diberikan sistem:

    • x + y + z = 6
    • 2x − y + 3z = 14
    • −x + 4y − z = −2

    Metode Eliminasi:

    1. Eliminasi x: tambahkan persamaan pertama dan ketiga: (x − x) + (y + 4y) + (z − z) = 6 − 2 → 5y = 4 → y = 4/5
    2. Substitusi y ke persamaan pertama: x + 4/5 + z = 6 → x + z = 26/5
    3. Substitusi y ke persamaan kedua: 2x − 4/5 + 3z = 14 → 2x + 3z = 74/5
    4. Selesaikan sistem dua variabel: x + z = 26/5 dan 2x + 3z = 74/5 → x = 12/5, z = 14/5
  • Rumus Cepat SPLDV (2x2):
    Jika persamaan: a1·x + b1·y = c1 dan a2·x + b2·y = c2, maka:
    • x = (c1·b2 − c2·b1) / (a1·b2 − a2·b1)
    • y = (a1·c2 − a2·c1) / (a1·b2 − a2·b1)

Materi Barisan & Deret

  • Barisan Aritmetika (Un): Un = a + (n−1)d
  • Jumlah n suku pertama (Sn): Sn = n/2 · (a + Un)
  • Barisan Geometri (Un): Un = a·rn−1
  • Jumlah n suku pertama (Sn): Sn = a·(rn − 1)/(r−1), r≠1

Materi Trigonometri

  • sin²θ + cos²θ = 1
  • tanθ = sinθ / cosθ
  • sin(a ± b) = sin a cos b ± cos a sin b
  • cos(a ± b) = cos a cos b ∓ sin a sin b
  • Identitas dasar: sin30°=1/2, cos30°=√3/2, tan45°=1
